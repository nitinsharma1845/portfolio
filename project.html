Blog Go – React Blogging Platform

BlogGo is a full-featured blogging platform built using React and Redux Toolkit, where users can create, update, and manage blog posts with a clean and intuitive interface. The application allows users to publish, edit, deactivate, or reactivate blog posts in real-time using Appwrite as the backend for authentication and database management.

Key features include a rich text editor powered by TinyMCE, real-time feedback via React Toastify, and dynamic HTML rendering using HTML Parser. The interface is styled with Tailwind CSS for responsive, utility-first design.

Tech Stack & Features

  • React – Component-based UI framework
  • Redux Toolkit – State management and API handling
  • Tailwind CSS – Utility-first styling
  • Appwrite – Backend as a service for database & auth
  • TinyMCE – Rich text editor for blog content
  • React Toastify – Clean, customizable toast notifications
  • HTML Parser – Secure rendering of HTML content
  • CRUD Operations – Post, update, delete, activate/deactivate blogs
  • Responsive Design – Fully mobile-friendly layout

Paste App – Note Saving Application

Paste App is a simple and efficient note-taking application built with React that allows users to create, manage, and search notes directly in the browser. All data is stored in Local Storage, ensuring offline functionality and quick access without requiring a backend server.

The application provides full CRUD operations, letting users add, edit, delete, and search notes with ease. It uses Context API for global state management, React Toastify for real-time user feedback, and is styled with Tailwind CSS for a clean and responsive UI.

Tech Stack & Features

  • React – SPA architecture and component-based UI
  • Local Storage – Data persistence without a backend
  • Context API – Global state management
  • Tailwind CSS – Modern utility-first styling
  • React Toastify – Instant feedback on user actions
  • CRUD Operations – Create, edit, delete, and view notes
  • Search Functionality – Filter notes by keyword
  • Responsive Design – Fully mobile-friendly interface
